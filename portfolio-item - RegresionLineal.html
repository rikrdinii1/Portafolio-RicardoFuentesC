Proyecto Realizado durante el Bootcamp de Data Analytics en Ironhack

Generación de Predicciones con Regresión Lineal Múltiple

Proyecto de fin de modulo 2 del Bootcam de Data Analytics en Ironhack

Durante este modulo se ve el tema de Estadísticas y Probabilidad.

Mi proyecto consiste en analizar un Data Set obtenido de Kaggle, el cual contiene las variables y precios de casas en venta en USA.

Se identifican las variables mas importantes, las cuales afectan el precio de la casa y se genera un Modelo de Regresión Lineal para predecir el precio de las casas con diferentes variables.
